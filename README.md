# Aditya Singh - Portfolio Website

A beautiful responsive portfolio website for Aditya Singh, created with HTML, CSS, JavaScript, and Node.js.

## Features

- Responsive design that works on all devices
- Modern and clean UI with animations
- Interactive elements using JavaScript
- Particle.js background animation
- Typed.js for dynamic text animation
- Contact form
- Skills showcase with progress bars
- Project section with placeholders for future projects
- Node.js Express server to serve the portfolio

## Technologies Used

- HTML5
- CSS3 (with responsive design)
- JavaScript (ES6+)
- Node.js
- Express.js
- Particle.js
- Typed.js
- Font Awesome
- Google Fonts

## Getting Started

### Prerequisites

To run this website locally, you need to have Node.js and npm installed on your computer.

### Installation

1. Clone this repository or download the source code
2. Navigate to the project directory

```bash
cd portfolio
```

3. Install the dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```

5. Open your browser and visit `http://localhost:3000`

### Running in Production

For production deployment, you can use:

```bash
npm start
```

## Customization

You can easily customize this portfolio by modifying:

- Content in `index.html`
- Styles in `public/css/style.css`
- Functionality in `public/js/script.js`
- Server settings in `server.js`
